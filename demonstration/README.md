# AI旅游助手 - 前端界面演示

这是一个AI旅游助手项目的前端界面演示，基于之前设计文档实现的所有页面。包含完整的功能界面和交互效果，方便开发参考和调整。

```

└── README.md               # 说明文档
```

## 🎨 CSS架构

### 自定义属性（CSS Variables）

```css
:root {
  --color-primary: #3b82f6;        /* 主色 */
  --color-secondary: #8b5cf6;      /* 辅助色 */
  --color-success: #10b981;        /* 成功色 */
  --color-warning: #f59e0b;        /* 警告色 */
  --color-error: #ef4444;          /* 错误色 */

  /* 背景色 */
  --color-bg-primary: #ffffff;
  --color-bg-secondary: #f9fafb;

  /* 文本色 */
  --color-text-primary: #111827;
  --color-text-secondary: #6b7280;

  /* 阴影 */
  --shadow-sm: 0 1px 2px 0 rgb(0 0 0 / 0.05);
  --shadow-md: 0 4px 6px -1px rgb(0 0 0 / 0.1);
  --shadow-lg: 0 10px 15px -3px rgb(0 0 0 / 0.1);
}
```

### 组件化样式

- **按钮系统**：`.btn`, `.btn-primary`, `.btn-outline`等
- **表单组件**：`.input`, `.form-group`, `.form-label`等
- **卡片组件**：`.card`, `.card-header`, `.card-body`等
- **导航组件**：`.sidebar`, `.sidebar-item`等
- **消息组件**：`.chat-message`, `.chat-bubble`等

### 响应式断点

- `sm`: 640px+
- `md`: 768px+
- `lg`: 1024px+
- `xl`: 1280px+
- `2xl`: 1536px+

## 🚀 使用方法

### 1. 直接打开

直接在浏览器中打开任意HTML文件即可查看：

```bash
# 登录页面
open login.html

# 注册页面
open register.html

# 设置页面
open settings.html
```

### 2. 本地服务器

使用Python简单HTTP服务器：

```bash
# Python 3
python -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080

# 然后访问 http://localhost:8080
```

### 3. Live Server (VS Code)

安装Live Server扩展，右键HTML文件选择"Open with Live Server"。

## 📝 开发指南

### 添加新页面

1. 复制现有页面模板
2. 修改页面标题和内容
3. 在侧边栏添加导航链接
4. 更新导航激活状态

### 自定义主题

编辑 `css/styles.css` 中的CSS变量：

```css
:root {
  --color-primary: #your-color;
  --color-secondary: #your-color;
}
```

### 修改样式

1. **使用Tailwind**：优先使用Tailwind类名
2. **自定义CSS**：写在 `<style>` 标签或独立CSS文件
3. **组件样式**：遵循BEM命名规范

### JavaScript最佳实践

1. **模块化**：按功能拆分代码
2. **事件委托**：优化事件处理
3. **防抖节流**：处理频繁触发
4. **错误处理**：添加try-catch
5. **注释完整**：便于维护
